# 02. Web Server

This ESP32 project serves a simple web page over its own Wi-Fi network.

## Goal
Learn how ESP32 can host a local web server and respond to requests from a browser.

## What I practiced
- ESP32 Wi-Fi Access Point mode
- local IP access in browser
- basic web server logic
- browser-based interaction with ESP32

## Process
1. Opened Arduino IDE in Linux Mint.
2. Selected the correct ESP32 board and port.
3. Uploaded the ESP32 web server sketch.
4. Connected the phone or laptop to the ESP32 Wi-Fi network.
5. Opened the ESP32 local IP address in a browser.
6. Verified that the page loaded successfully.
7. Tested browser interaction with the ESP32.

## Result
The ESP32 successfully hosted a local web page that could be opened from a phone or laptop connected to its Wi-Fi network.
